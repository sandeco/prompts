# Antigravity Kit — Guia de Instalação

**O que é**: Kit oficial do Google Antigravity para inicializar projetos com estrutura de agentes, skills e workflows prontos para uso.

---

## Instalação

**Pré-requisitos**: Node.js com npx — verifique: `node --version` e `npx --version`

Execute no **diretório raiz do projeto**:

```bash
npx @vudovn/ag-kit init
```

O CLI interativo irá guiar a configuração inicial do projeto.

---

## Próximos passos após instalação

1. Siga os prompts do CLI para configurar o projeto
2. Abra o projeto no Google Antigravity IDE
3. Use `/` para ver os workflows e skills disponíveis
