# BMad Method — Guia de Instalação

**O que é**: Breakthrough Method for Agile AI-Driven Development. Framework completo de desenvolvimento ágil com IA, com agentes especializados, workflows guiados e planejamento inteligente adaptativo.

**Repositório oficial**: https://github.com/bmad-code-org/BMAD-METHOD
**Documentação**: https://docs.bmad-method.org/

---

## Pré-requisitos

- Node.js instalado (verifique: `node --version`)
- npx disponível (vem com Node.js — verifique: `npx --version`)

---

## Instalação

Execute no **diretório raiz do projeto**:

```bash
npx bmad-method install
```

O instalador irá fazer perguntas interativas sobre o projeto. Siga os prompts.

---

## O que é criado

Após a instalação, o BMad cria uma estrutura de agentes e workflows no projeto, geralmente dentro de uma pasta `.bmad/` ou similar com:
- Agentes especializados (PM, arquiteto, desenvolvedor, QA, etc.)
- Workflows de desenvolvimento
- Templates de documentação

---

## Próximos passos após instalação

1. Abra o projeto no seu AI IDE (Claude Code, Cursor, Windsurf, etc.)
2. Use `bmad-help` para descobrir o que fazer a seguir
3. Exemplo de uso: peça ao agente "bmad-help I want to start a new feature"
